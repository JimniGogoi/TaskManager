# TaskManager
Team Task Manager (Full-Stack) Key Features:  Auth: Secure Login/Signup system.  Projects: Create/manage multiple workstreams.  Task Tracking: Toggle status (Pending/Completed) &amp; assign due dates.  Overdue Logic: Red flags for past-due tasks.  Tech Stack:  Frontend: React + Bootstrap.  Backend: Flask + SQLite.  Deploy: Live on Railway.
